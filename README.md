# print-feed

## Environment variables
- Add API key to `.env` file as `HARVARD_ART_API_KEY`

## Development mode
- `npm dev`
- visit `http://localhost:3000`

## Production mode
- `npm run build`
- `npm start`
- visit `http://localhost:3000`

## Tests
- `npm test`

## Visual Tests
- `npm run test:debug`
